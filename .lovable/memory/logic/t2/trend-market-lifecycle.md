# Memory: logic/t2/trend-market-lifecycle
Updated: now

트렌드 예측 마켓은 매일 00:00(KST) 추적(track) 시점을 기준으로 24시간 주기로 운영됨. 00:00 추적이 완료되면 해당 키워드의 점수가 시장의 베이스라인(Initial Influence)으로 설정되며 마켓이 자동으로 오픈됨. 마켓 활성화 조건은 키워드가 'active' 상태이고 baseline_score가 0보다 큰 경우임. 유저는 다음 추적 2시간 전인 22:00(KST)까지 예측할 수 있음. 24시간 후 다음 추적 주기 완료 시 성장률(Trend Score 변화량)에 따라 정산이 이루어짐. 추적 실패 시에는 참여 보상(10T)만 지급됨. 활성 상태의 키워드는 매일 새로운 주기의 마켓이 반복적으로 열림.

## 리워드 구조 (배팅 제거)
- 유저는 K-Token을 걸지 않음 (무료 참여)
- 구간 선택만으로 예측 참여
- 리워드: 소폭(100T), 강세(300T), 폭발(1,000T)
- 틀려도 참여 보상: 10T
- 마켓 미생성(추적 전) 상태: "곧 열려요!" 문구 + 남은 시간 표시
