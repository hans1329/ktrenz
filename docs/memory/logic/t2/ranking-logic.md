# Ranking Logic (T2 Treemap)

트리맵 박스 크기는 **단일 로그 변환** (`Math.log1p(metric)`)을 적용하여 점수 차이를 자연스럽게 반영함. 이전의 이중 로그(double-log) + cap(4) 방식은 과도한 평탄화를 유발하여 모든 박스가 비슷한 크기로 렌더링되는 문제가 있었음.

랭크 기반 배율: 1위 ×1.55, 2위 ×1.5, 3위 ×1.45, ~5위 ×1.4, ~11위 ×1.35, ~24위 ×1.3, ~39위 ×1.25, 나머지 ×1.2.

폰트 크기: 상위 3위 18–36px, 상위 20위 14–28px, 기타 12–24px. 아티스트명은 키워드 대비 0.7배(최소 9px).
